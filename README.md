# Hamza Abu Saado - Portfolio

A professional portfolio website showcasing my skills, experience, and projects in IT support, system administration, and network engineering.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Contact form, smooth scrolling, dynamic navigation
- **Downloadable CV**: Quick access to resume PDF
- **Skill Categories**: Organized technical skills by domain
- **Project Showcase**: Detailed homelab projects with tags and descriptions

## Sections

1. **Hero**: Introduction and quick contact
2. **About**: Professional summary and highlights
3. **Experience**: Timeline of professional work
4. **Projects**: Showcase of homelab and personal projects
5. **Skills**: Technical skills organized by category
6. **Contact**: Contact form and professional links

## Technologies Used

- HTML5 semantic structure
- CSS3 with modern features (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript (no dependencies)
- Font Awesome icons
- Google Fonts (Inter)

## File Structure


## Customization

To customize the portfolio:

1. **Personal Info**: Update contact details in the contact section
2. **Projects**: Add/modify projects in the projects section
3. **Skills**: Update skills categories and items
4. **Colors**: Modify CSS custom properties in `:root` section
5. **Resume**: Add your PDF resume as `CV.pdf`
